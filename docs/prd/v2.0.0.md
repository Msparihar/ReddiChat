# PRD: ReddiChat v2.0.0 — Community Intelligence

**Status:** Shipped
**Date:** 2026-03-28

## Version Goal

Transform ReddiChat from "AI chat that searches Reddit" into a **Community Intelligence platform** — where users get sentiment analysis, cross-subreddit comparisons, trending topics, and thread summaries that no other AI chat provides.

## Features

### 1. Sentiment Analysis & Community Consensus
- When users ask "What do people think about X?", analyze Reddit posts/comments to show:
  - Sentiment distribution (positive/negative/neutral %)
  - Key agreement and disagreement points
  - Subreddit-specific viewpoints
- New AI tool: `analyze_sentiment` — searches Reddit, scores posts, returns structured sentiment data
- Visual sentiment bar in chat response
- **Files:** New `frontend/lib/ai/tools/sentiment.ts`, new UI component

### 2. Cross-Subreddit Comparison
- Compare how different communities view the same topic
- Example: "How do r/investing vs r/wallstreetbets view crypto?"
- Returns: viewpoint differences, risk tolerance, terminology differences per community
- New AI tool: `compare_subreddits` — searches same query across multiple subreddits, contrasts results
- **Files:** New `frontend/lib/ai/tools/compare.ts`

### 3. Trending Topics & Emerging Discussions
- Surface what's trending on Reddit right now
- "What's hot in r/technology today?"
- Ranks by engagement velocity (upvotes + comments per hour)
- Shows emerging vs established trends
- New AI tool: `get_trending` — fetches hot/new posts, calculates velocity
- **Files:** New `frontend/lib/ai/tools/trending.ts`

### 4. TLDR Generator for Long Threads
- Detect Reddit links in user messages
- Auto-fetch thread and top comments
- Generate structured summary: opposing viewpoints, consensus, key facts, most helpful comment
- Inline thread preview in chat
- New AI tool: `summarize_thread` — fetches thread data, produces structured TLDR
- **Files:** New `frontend/lib/ai/tools/summarize.ts`, new UI component

### 5. Subreddit Deep-Dive Pages
- New route: `/r/[subreddit]` — community profile page
- Shows: description, recent hot topics, community culture analysis, common debates, key contributors
- "Ask about this subreddit" quick action that opens chat with context
- **Files:** New `frontend/app/r/[subreddit]/page.tsx`, new components

### 6. Rich Source Previews
- Show thumbnails for image posts in source cards
- Link preview (title + description) for URL posts
- Expandable post preview without leaving the chat
- **Files:** `frontend/components/chat/RedditSource.tsx`

### 7. Enhanced System Prompt
- Update AI system prompt to leverage new tools intelligently
- AI should proactively suggest sentiment analysis when questions are opinion-based
- AI should compare subreddits when multiple communities are relevant
- **File:** `frontend/lib/ai/system-prompt.ts`

## Out of Scope

- User-facing API (B2B)
- Monetization / billing
- Reddit profile credibility scoring (v2.1.0)
- Real-time notifications / alerts
- Browser extension

## Done Criteria

- [ ] Sentiment analysis tool returns structured data with visual bar in chat
- [ ] Cross-subreddit comparison works for 2-5 subreddits
- [ ] Trending topics shows velocity-ranked posts for any subreddit
- [ ] Reddit links in messages auto-generate TLDR summaries
- [ ] `/r/[subreddit]` page shows community profile with culture analysis
- [ ] Source cards show thumbnails and expandable previews
- [ ] System prompt uses new tools intelligently based on question type
- [ ] All new tools have error handling and rate-limit awareness
- [ ] TypeScript compiles without errors
