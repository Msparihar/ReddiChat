# PRD: ReddiChat v1.0.0

**Status:** Shipped
**Date:** 2025 (initial release)

## Version Goal

Build a Reddit-powered AI chat application with social authentication, file uploads, and real-time streaming responses.

## Features

- **AI Chat with Streaming** — Real-time SSE streaming responses using Gemini 2.0 Flash via AI SDK
- **Reddit Search Tool** — AI can search Reddit posts/comments via snoowrap
- **Web Search Tool** — AI can search the web via Tavily
- **Social Auth** — Google and GitHub OAuth via Better Auth
- **Conversation History** — Persistent threads stored in PostgreSQL via Drizzle ORM
- **File Uploads** — Image and document uploads to S3 with message attachments
- **Login Notifications** — Email alerts on new sign-ins via Nodemailer
- **Landing Page** — Redesigned landing with inline auth and user lookup
- **Dark/Light Theme** — Theme toggle with system preference detection

## Technical Scope

- **Framework:** Next.js 15 (App Router, frontend-only architecture)
- **State:** Zustand for client state, React Query for server data
- **AI:** AI SDK v6 with Google Gemini provider (`@ai-sdk/google`)
- **Auth:** Better Auth with social OAuth providers
- **Database:** PostgreSQL + Drizzle ORM (conversations, messages, file_attachments, message_attachments)
- **Storage:** AWS S3 for file uploads
- **Styling:** Tailwind CSS + Radix UI primitives

## Key Files

- `frontend/lib/ai/agent.ts` — AI streaming and tool orchestration
- `frontend/lib/ai/tools/` — Reddit search and web search tools
- `frontend/stores/chat-store.ts` — Chat state management
- `frontend/lib/streaming.ts` — SSE stream parser
- `frontend/app/api/chat/stream/route.ts` — Chat API endpoint
- `frontend/lib/auth/index.ts` — Auth config with login hooks
- `frontend/lib/email/` — Email service

## Out of Scope

- Multi-model support
- User-selectable AI models
- Rate limiting / usage tracking
- Mobile-optimized UI
