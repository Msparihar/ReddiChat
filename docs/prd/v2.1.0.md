# PRD: ReddiChat v2.1.0 — Monetization & Growth

**Status:** Draft
**Date:** 2026-03-28

## Version Goal

Add usage tracking, tiered access, and billing to turn ReddiChat into a revenue-generating product.

## Features

### 1. Usage Tracking
- Track per-user: messages sent, AI tokens consumed, tools used, file uploads
- Store in dedicated `usage` table with daily aggregation
- Dashboard: "You've used X of Y messages this month"

### 2. Tiered Access (Freemium)
- **Free tier:** 20 messages/day, Gemini 2.5 Flash only, basic Reddit search
- **Pro tier ($8/month):** Unlimited messages, all models, sentiment analysis, trending topics, TLDR, export
- **Team tier ($25/month):** Everything in Pro + subreddit monitoring, API access, priority support
- Gate premium features behind tier check middleware

### 3. Billing Integration
- Stripe Checkout for subscription management
- Webhook handler for payment events (subscription created/cancelled/failed)
- Billing settings page: current plan, usage, upgrade/downgrade, cancel

### 4. Reddit Profile Credibility Analysis
- Expertise scoring: karma density per subreddit
- Perspective bias detection
- "Best contributions" highlight
- Credibility context in AI responses

### 5. Observability Stack
- OpenTelemetry integration for traces and metrics
- Log aggregation (Axiom or similar)
- Error tracking (Sentry)
- Uptime monitoring with alerts

### 6. Public API (B2B)
- REST API for programmatic access to:
  - Sentiment analysis
  - Trending topics
  - Subreddit comparison
- API key management in user dashboard
- Rate limits per tier

## Out of Scope

- Mobile app (native)
- Browser extension
- Enterprise SSO
- White-label offering
- Real-time Slack/Discord integration

## Done Criteria

- [ ] Usage tracked and displayed per user
- [ ] Free/Pro/Team tiers enforced on all endpoints
- [ ] Stripe billing functional (subscribe, cancel, upgrade)
- [ ] Profile credibility analysis working in chat
- [ ] Structured logs and metrics flowing to aggregation service
- [ ] Public API documented and gated by API key
- [ ] TypeScript compiles without errors
