# PRD: ReddiChat v1.3.0 — UX Polish

**Status:** Draft
**Date:** 2026-03-28

## Version Goal

Polish the user experience with essential UX improvements — copy responses, mobile layout, conversation search/export, streaming feedback, and keyboard shortcuts.

## Features

### 1. Copy Button on AI Responses
- Add copy-to-clipboard button below each assistant message
- Show toast confirmation on copy
- **File:** `frontend/components/chat/MessageList.tsx`

### 2. Mobile-Responsive Chat Layout
- Sidebar: hamburger menu on <768px, full-width overlay when open
- Increase touch targets to 44px minimum (WCAG)
- Stack Reddit source cards vertically on mobile
- Mobile-friendly model selector
- **Files:** `frontend/app/(chat)/layout.tsx`, `frontend/components/chat/MessageInput.tsx`, `frontend/components/chat/RedditSource.tsx`, `frontend/components/layout/Sidebar.tsx`

### 3. Conversation Search
- Search input in sidebar to filter conversations by title/content
- Query API with search parameter
- Highlight matching text in results
- **Files:** `frontend/components/layout/Sidebar.tsx`, `frontend/app/api/chat/history/route.ts`

### 4. Conversation Export
- Export conversation as Markdown file
- Export as shareable read-only link (optional, if time allows)
- **Files:** New export utility, sidebar action button

### 5. Streaming Feedback Phases
- Show operation phases during AI response: "Searching Reddit..." → "Analyzing sources..." → "Generating response..."
- Display response time after completion
- Timeout warning if response takes >15s
- **Files:** `frontend/stores/chat-store.ts`, `frontend/components/chat/MessageList.tsx`

### 6. Keyboard Shortcuts
- `Cmd/Ctrl + K` — search conversations
- `Cmd/Ctrl + N` — new chat
- `Cmd/Ctrl + /` — show shortcuts help
- `Escape` — close modals/overlays
- **Files:** New `frontend/hooks/useKeyboardShortcuts.ts`

### 7. Better Error Messages
- Show categorized errors: rate limit, network, auth, AI provider
- "Retry" button with backoff indicator
- Distinguish user-fixable vs server-side errors
- **Files:** `frontend/stores/chat-store.ts`, `frontend/components/chat/MessageList.tsx`

### 8. Accessibility Improvements
- Add missing `aria-label` attributes on all interactive elements
- Focus-visible states on buttons and links
- `aria-busy` on streaming messages
- Proper table `scope` attributes in markdown renderer

## Out of Scope

- Rich source previews / thumbnails (v2.0.0)
- Subreddit browsing (v2.0.0)
- Conversation pinning/favorites
- Multi-tab sync
- Analytics/usage tracking

## Done Criteria

- [ ] Copy button visible on all assistant messages, toast on click
- [ ] Chat UI fully usable on mobile (iPhone SE to iPad)
- [ ] Conversations searchable by title from sidebar
- [ ] Export to Markdown works for any conversation
- [ ] Streaming shows distinct phases (search, analyze, generate)
- [ ] All keyboard shortcuts functional
- [ ] Error messages categorized with actionable text
- [ ] All interactive elements have aria-labels
- [ ] TypeScript compiles without errors
