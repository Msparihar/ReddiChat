# PRD: ReddiChat v1.7.0 — Design System Overhaul

**Status:** In Progress
**Date:** 2026-04-05

## Version Goal

Apply a cohesive, premium design system anchored by Reddit Orangered (`#ff4500`) with Apple-inspired shadows, warm typography, and systematic color tokens — transforming ReddiChat from "functional chat app" to "polished product."

## Features

### 1. CSS Token System
- Replace default shadcn HSL variables with ReddiChat-specific `--rc-*` CSS custom properties
- Warm light theme surfaces (`#f8f7f6`, `#f0efed`) instead of cold grays
- Warm dark theme surfaces (`#0c0c0d`, `#141416`, `#1b1b1e`) instead of generic blacks
- Semi-transparent `rgba` borders replacing hard hex borders
- Apple-inspired shadow tokens: resting (`2px 4px 24px`), hover (`3px 5px 30px`), elevated, popover
- Orange focus ring system (`rgba(255, 69, 0, 0.2)`)
- **Files:** `frontend/globals.css`, `frontend/tailwind.config.ts`

### 2. Typography Consolidation
- Remove 4 unused Google Font imports (Space Grotesk, Outfit, DM Sans, Syne)
- Keep Plus Jakarta Sans (display + UI) and JetBrains Mono (code/technical)
- Apply negative letter-spacing on headings (-1.5px to -2px)
- Enable OpenType `"ss01"` for cleaner letterforms
- Update `fontFamily` in Tailwind config
- **Files:** `frontend/globals.css`, `frontend/tailwind.config.ts`

### 3. Brand Color Unification
- Replace all `purple-600`, `purple-500`, `purple-*` with Orangered variants
- "New Chat" button: `#ff4500` background instead of purple
- Active category pills: Orangered tint instead of purple
- Sidebar active thread: Orangered left border + soft tint
- "Upgrade to Pro" pill: Orangered gradient instead of purple-to-blue
- Welcome category icons: all secondary text color, Orangered only on active
- **Files:** `frontend/components/layout/Sidebar.tsx`, `frontend/components/chat/WelcomeScreen.tsx`, `frontend/components/chat/MessageInput.tsx`, `frontend/components/layout/ProfileDropdown.tsx`, `frontend/components/layout/ConversationHistory.tsx`

### 4. Shadow & Depth System
- Light theme cards: Apple-style diffused shadow with hover lift transition
- Dark theme cards: border luminance instead of shadows
- Prompt chips: shadow-based on light, border-based on dark
- Message input: elevated treatment with subtle shadow + border
- Smooth hover transitions (0.3s ease on box-shadow)
- **Files:** All component files using cards, inputs, and interactive surfaces

### 5. Component Refinement
- Message input: 12px radius, elevated shadow, orange focus ring
- Sidebar: warm surface colors, refined border treatment
- Welcome screen: Apple-shadow prompt chips, compressed heading
- Profile dropdown and popovers: popover shadow token
- Scrollbar: keep orange gradient (already on-brand)
- **Files:** All component files in `frontend/components/`
